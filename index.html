Fedora Repository Camel Component Developer Documentation

Fedora is a robust, modular, open source repository system for the management and dissemination of digital content. Camel is a widely used event-oriented framework for integrating applications. The fcrepo-camel component allows developers to write custom Camel-based routes that interact with a Fedora repository. Camel is particularly well suited for developing microservices or implementing a distributed, service-oriented architecture.
